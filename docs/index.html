/03 Architecture

Symmetric Desktop & Web Core

The main helper components reside inside the src/main subdirectory structure. The documentation is self-contained inside the docs directory, enabling clean static server hosting.

Java Core src/main/java/com/naik/NaikApp.java
Command Manager src/main/java/com/naik/commands/CommandProcessor.java
