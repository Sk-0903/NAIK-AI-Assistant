GitHub Pages

Deploy the docs folder

  1. 01 Create a GitHub repository and push this project.
  2. 02 Open repository Settings, then select Pages.
  3. 03 Choose "Deploy from branch" under build and deployment.
  4. 04 Select your branch and set the directory folder to /docs.
  5. 05 Save the settings and wait for GitHub to publish the site.
